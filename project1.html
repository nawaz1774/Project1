Section 1 : Statistical Test

1.1 Which statistical test did you use to analyse the NYC subway data? Did you use a one-tail or a two-tail P value? What is the null hypothesis? What is your p-critical value?

Statistical test details:

The statistical test used for analysis is Mann-Whitney U test.

Choice of Tailed Test:

Since the Mann-Whitney U computes one tailed statistic we also had to observe the means of the 2 NYC Subway data sets. The two data sets here are Ridership when rains and Ridership when it does not rain. The general assumption is that people will prefer subway when it does not rains.

  • The mean calculated for the Data set when no rains turned out to be 1105.4463767458733 (NRM).
  • The mean calculated for the Data set when rain turned out to be 1090.278780151855 (RM).
  • Since NRM is greater that RM we are trying to prove NRM is significantly greater than RM (One - Tailed Test)

NULL Hypothesis:

The NULL Hypothesis in this case is that Mean of the Ridership values when no rain (NRM) is not significantly higher (One - Tailed Test) than mean of the Ridership when it rains (RM). To put it in another way NULL Hypothesis is that people's ridership on NYC Subway has no dependency on rain conditions.

P-critical value:

In this case we have set p-critical value to be 0.05. P-critical value is a value that is used as threshold to decide whether to NULL hypothesis can be rejected or retained.

1.2 Why is this statistical test applicable to the dataset? In particular, consider the assumptions that the test is making about the distribution of ridership in the two samples.

Statistical test:

  • The statistical test chosen for the analysis of the NYC subway data was Mann-Whitney U test.
  • Mann-Whitney U is a non-parametric test that does not assume any particular probability distribution for the underlying data.
  • Upon plotting the data it was observed that data was positively skewed. Refer the below histogram.
  • Since the data provided is not normally distributed Mann-Whitney U test was chosen.

1.3 What results did you get from this statistical test? These should include the following numerical values: p-values, as well as the means for each of the two samples under test.

Below are the calculated values:

  • The mean calculated for the Data set when no rains turned out to be 1105.4463767458733 (NRM).
  • The mean calculated for the Data set when rain turned out to be 1090.278780151855 (RM)
  • Probability value calculated with Mann Whitney U test turned out to be 0.025.

1.4 What is the significance and interpretation of these results?

Below are the interpretations:

  • The mean of ridership calculated for when it does not rain is higher than when it rains. So the general assumption would be that people prefer taking NYC Subway when it does not rain. This assumption formulates our Alternate Hypothesis.
  • In order to quantify our assumption a statistical test called Mann Whitney U was used.
  • The Alpha value (p-critical) value has been chosen to be 0.05. Any p-value less than this, signifies that a particular incident did not happen by chance. There was some kind of intervention that caused these results.
  • In this case we calculated the probability value to be 0.025 which is less than 0.05 (p-critical) which signifies that this difference in means of ridership when it rains to when it does not rain did not occur by chance rather occurred due some kind of intervention. In this case rain on that particular day.

Approach used:

  • Gradient descent algorithm was used to determine the weights or coefficients of our regression model.
  • In this method multiple values for theta's are applied to the cost function calculation across multiple iterations till we find those values for which the cost function is at its minimum.
  • The final goal is to calculate the values of Thetas for which the error or difference between a predicted value and actual values are at its minimum across all the data points.

The features or input values that were used for this regression model are:

4.2 What analyses lead you to this conclusion? You should use results from both your statistical tests and your linear regression to support your analysis.

Brief Description of the Data:

  • The data provided is the data that has been collected at various NYC Subway Turnstiles.
  • Each row in the data represents a point a time along with details such as number of entries and exits that have occurred in the last one hour.

Statistical Tests Performed on this Data:

  • Two statistical test listed below have been performed on this data:
    • Mann-Whitney U test
    • Linear Regression with Gradient Descent.

Mann-Whitney U test:

  • This statistical test has been performed to answer question "What is the effect of Rain on the NYC Subway ridership?"
  • Mann-Whitney U is a statistical test that quantifies the likelihood of a particular claim over the other. That's why it is also call Hypothesis testing.
  • In Hypothesis testing we either retain or reject a NULL Hypothesis.
  • NULL Hypothesis in this case is that there is no dependency of rain on ridership of NYC Subway.
  • Results and Observations
    • The mean of ridership when it did not rain was calculated to be 1105.45
    • The mean of ridership when it rained was calculated to be 1090.28
    • The mean when it did not rain is higher than when it rained.
    • In order to quantify this we performed Mann-Whitney U with a critical value of 0.05.
    • The probability value of Mann-Whitney U after passing in these two data sets was calculated to be 0.025.
    • Since the probability value calculated is well below the critical value we can conclude that higher value for the mean "when no rain"" did not occur by chance, but due to some kind of intervention. In this case rain.

Linear Regression with Gradient Descent:

This is a statistical model that is used for making predictions about an occurrence of an event based on a previous study that has occurred on an existing data set for which we have results. In this case we are trying to predict the NYC Subway ridership choice of New Yorkers taking into considerations some factors (atmospheric conditions in this case). The inputs that have been provided to this model are rain, precipitation, hour of the day and mean temperature. Below are the list inputs along with their corresponding Weights or Coefficients for this statistical model.

  • Rain : 2.92398062e+00
  • Precipitation : 1.46526720e+01
  • Hour : 4.67708502e+02
  • Mean temp : -6.22179395e+01

Weights or Coefficients for an input parameter determine effect it has on output variable. Higher the value of the coefficient stronger the relationship it has with the output parameter. Stronger the relationship the better it can be used to predict the output. These coefficients are calculated using the Gradient Descent algorithm.

Looking at the coefficient values for the above parameters Hour of the day or Time of the day seems to have strongest relationship to the output parameter "Subway Ridership". Second highest relationship seems to be with parameter "occurrence of rain".

After observing the above results from the both the Statistical tests it can be concluded that Occurrence of rain is definitely a contributing factor that determines NYC Subway Ridership.
